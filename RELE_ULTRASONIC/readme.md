# 🤖 ESP32 Autonomous Robot (Ultrasonic + BNO055 + FreeRTOS)

## 📌 Описание проекта

Данный проект реализует автономное движение робота на базе ESP32 с использованием:

* 3 ультразвуковых датчиков (лево, центр, право)
* гироскопа BNO055
* 2 реле управления двигателями
* FreeRTOS (настоящая многозадачность)

Робот способен:

* обнаруживать препятствия
* обходить их
* возвращаться на исходный курс
* останавливаться при невозможности движения
* работать в ручном режиме (Flag)

---

## ⚙️ Используемое оборудование

* ESP32 DevKit V1
* Ультразвуковые датчики (HC-SR04 или аналог)
* BNO055 (гироскоп + компас + акселерометр)
* Релейный модуль (2 канала)
* DC моторы / драйвер

### Управление

| Назначение  | Пин |
| ----------- | --- |

## 🧠 Как едет робот

### 📍 Основной принцип

Робот использует:

* **датчики → чтобы понять где препятствие**
* **гироскоп → чтобы контролировать направление**

---

## 🚗 Движение вперёд

Если центральный датчик свободен:

→ включаются оба реле

```text
RELAY_L = ON
RELAY_R = ON
```

➡️ робот едет прямо

---

## 🚧 Обнаружение препятствия

Если впереди препятствие:

1. робот проверяет правую сторону
2. если справа свободно → поворачивает вправо
3. если нет → проверяет левую сторону

---

## ↪️ Поворот

### Вправо:

```text
RELAY_L = ON
RELAY_R = OFF
```

➡️ левое колесо крутится → поворот вправо

---

### Влево:

```text
RELAY_L = OFF
RELAY_R = ON
```

➡️ правое колесо крутится → поворот влево

---

## 🧭 Использование гироскопа

Гироскоп (BNO055) даёт угол:

```text
yaw = 0° ... 360°
```

---

### Алгоритм:

1. робот запоминает текущий угол (например 10°)
2. поворачивает (например до 40°)
3. объезжает препятствие
4. возвращается обратно к 10°

➡️ таким образом он **сохраняет направление движения**

---

## 🔄 Возврат на траекторию (гироскоп + время)

После обхода:

1. робот возвращает угол (гироскоп)
2. затем едет вперёд столько же времени, сколько ехал в обход

➡️ это позволяет приблизительно вернуться на прежнюю линию

---

## ⛔ Остановка

Если:

* слева занято
* справа занято
* впереди занято

➡️ робот останавливается

```text
RELAY_L = OFF
RELAY_R = OFF
```

и включает сигнал:

```text
SIGNAL_PIN = ON
```

---

## 🧰 Режим ручного управления (Flag)

Если:

```text
FlagPort = HIGH
```

➡️ робот:

* игнорирует датчики
* игнорирует гироскоп
* едет прямо

---

## 🧵 Архитектура (FreeRTOS)

Проект использует многозадачность:

### Task 1 — Датчики

* считывает расстояние

### Task 2 — Гироскоп

* считывает угол

### Task 3 — Логика

* принимает решения
* управляет движением

---

## 🔒 Синхронизация

Используется:

```text
Mutex (Semaphore)
```

для безопасного обмена данными между задачами

## ⚠️ Ограничения

* используется `pulseIn()` (блокирующий)
* нет энкодеров → позиция рассчитывается приблизительно
* возврат на линию не идеальный

```ini
    adafruit/Adafruit Unified Sensor
```
